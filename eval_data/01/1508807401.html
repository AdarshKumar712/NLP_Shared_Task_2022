Come installare Android usando solo il browser web

Google presenta un nuovo strumento che consentirà agli sviluppatori di aggiornare la ROM dei dispositivi Pixel e delle reference board con una procedura molto più semplice e rapida rispetto a quella abituale.

  • Tweet

Google ha appena presentato il nuovo strumento chiamato Android Flash Tool che aiuterà gli sviluppatori a installare le ultime versioni del sistema operativo sui dispositivi Pixel usando esclusivamente il browser web.

Usando il tool si possono caricare le più recenti build AOSP (Android Open Source Project) semplicemente collegandosi con la pagina web del progetto da sistemi Linux, macOS, Chrome OS e Windows (richiede un driver USB aggiuntivo il cui download viene automaticamente proposto).


Affinché tutto funzioni è necessario collegare il dispositivo mobile Android da aggiornare servendosi del cavo USB e usare un browser come Chrome oppure Edge basato su Chromium (versione 79 e successive) che supporti WebUSB.


Google osserva inoltre che sulla propria macchina fosse presente e avviato ADB (Android Debug Bridge), il servizio dovrà essere arrestato perché diversamente interferirebbe con il processo di flashing della ROM.


Dopo aver autorizzato il browser ad accedere alla lista delle chiavi crittografiche usate da ADB, si dovrà scegliere la build da usare, il dispositivo da aggiornare e Android Flash Tool penserà al resto.

Lo strumento è compatibile con i dispositivi Pixel 2, Pixel 2 XL, Pixel 3, Pixel 3 XL, Pixel 3a, Pixel 3a XL, Pixel 4, Pixel 4 XL così come con le reference board HiKey e HiKey 960 (vedere questa pagina).
